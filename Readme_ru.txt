Данное расширение выводит акционные продукты с таймером обратного отсчета и отображает процент скидки.
В модуле реализована возможность изменять количество товаров в строке на разных девайсах.
Так же добавлена возможность отключать отображение описания товара.
При окончании акции товар меняет состояние.

Не трогает файлы магазина.

Язык: en/ru.

Тестировалось на Opencart 2.0 и Opencart 2.1

Демо на сайте: http://svg.proitservice.in.ua/

Демо в админке: http://svg.proitservice.in.ua/admin Login: demo Pass: demo

Установка: 

Через меню "Модули" -> "Установка расширений" установить файл "special_countdown.ocmod.zip".

Обновить кэш модификаторов "Дополнения" -> "Модификаторы" Нажав на кнопку "Обновить"

Далее "Дополнения" -> "Модули" -> "Счетчик специальные предложения" Нажать на кнопку "Установить" и настроить модуль.
