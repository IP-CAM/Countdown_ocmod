This extension displays promotional products with a countdown timer and displays the discount percentage.
The module implements the ability to change the number of products in a row on different devices.
Also added the ability to disable the display of the item description.

At the end of the sale items changes state

Do not touch store files.

Language: en/ru.

Tested on OpenCart Opencart 2.0 and 2.1

Demo: http://svg.proitservice.in.ua/

Demo admin panel: http://svg.proitservice.in.ua/admin Login: demo Pass: demo

Installation:

The menu item "Extension" -> "Extension Installer" set file "special_countdown.ocmod.zip".


Clean Cache modifier "Extension" -> "Modifications " Click on the button "Refresh"

Next "Extension" -> "Модули" -> "Special Countdowns" Click on the button "Install" and configure the module.
